node-get-users [![Build Status](https://travis-ci.org/Jayin/node-get-users.svg)](https://travis-ci.org/Jayin/node-get-users)
=============
> get all users in you os except Windows

### install

```
npm install --save node-get-users
```

### Example

```
var getUser = require('./')
console.log(getUser())
// -> [ 'Guest', 'jayin' ]
```
